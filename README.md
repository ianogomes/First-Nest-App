## Description

This is a simple App made with the objective of applying my knowledge on basic NestJS architecture. The goal of the app is to store and retrieve simple string data (messages) into/from a JSON file.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
